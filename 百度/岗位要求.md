## 🆕 2026年最新更新（百度 AIDU 计划 + AI Agent 专项）

### AIDU 计划（2026届校招精英项目）

百度面向 2026 届博士生和硕士生的 AI 精英招募计划：
- **面向对象**：超级学霸、学术大神、竞赛高手、工程大牛
- **应届博士年薪**：100万起，顶尖人才不设上限
- **快速成长**：直接参与百度核心 AI 大模型项目（文心/知识增强大模型）

### AI Agent 算法工程师（2026 JD 更新）

**新增职责：**
- 负责 AI Agent 的感知-决策-执行闭环设计
- 推动 Agent 在**搜索、对话、办公、网盘文库、生活娱乐**等百度核心产品场景规模化应用
- 探索 **RAG 与 Agent 融合架构**（Agentic RAG），提升知识理解与行动协同
- 建设 Agent 评测体系：成功率、响应延迟、成本、用户体验多维度指标
- 研究 ReAct、AutoGPT、CoT 等前沿范式并工程化落地

**2026 新增要求：**
- 了解或实践过 **百度千帆 AppBuilder** Agent 开发平台
- 有 Agent 在搜索增强（Search-Augmented Agent）方向的经验优先
- 了解 **GraphRAG v2** 和最新 RAG 优化方案
