# 美团 AI Agent - 面试题 & 面经

## 面试特点
- 笔试包含大模型训练题 + 后端常规题
- 可能有 AI 面试官引导追问
- 重视 AI 在实际业务场景的应用

---

## 真实面试题

### AI 相关
1. 自我介绍中阐述 AI 大模型工具在学习和实践中的作用
2. 后端如何统计 token 用量、工具调用次数、重试次数？按业务场景和模型版本做聚合报表
3. 如何设计存储统计信息的数据库表（扩展性和性能）
4. 设计一个文本生成 HTTP 接口供业务方调用，支持流式返回
5. AI Agent 在美团业务中的应用场景

### 后端基础
6. TCP 粘包问题
7. IPv4 vs IPv6 区别与过渡策略
8. Java GC 机制及常见收集器，何时选 G1
9. 分库分表策略，高并发场景下如何选择
10. 消息队列原理，消费者跟不上生产者怎么办

### 系统设计
11. 设计项目管理系统（项目/任务/成员关系 + 数据库表结构）
12. 设计新闻推送系统（分类/推荐/实时推送）

### 行为面
13. 未来 3 年职业规划
14. 如何快速上手新项目
15. 如何顶住压力克服困难的经历

---

## 面试洞察
- 美团 Agent 岗偏工程落地，不是纯算法
- 系统设计题会结合 AI 场景（如设计文本生成接口）
- 后端基础不能丢，Java/分布式/数据库都会考
