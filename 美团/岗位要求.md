## 岗位一：大模型算法工程师

### 职位描述
探索LLM前沿研究，参与美团大模型基座研发。工作内容包括数据建设、模型预训练、模型压缩、推理优化、逻辑推理、Agent、MoE、Scaling Law和模型评估等。

### 技术要求
- 精通Python和PyTorch，有深度学习算法研发经验
- 深入理解Transformer架构和主流大模型
- 有预训练/微调/RLHF等大模型训练经验
- 熟悉分布式训练和推理优化技术
- 有推荐系统/搜索/NLP相关经验者优先
- 有顶会论文者优先

### 学历要求
- 硕士及以上学历，计算机/数学/统计相关专业
- 校招要求985/211，博士优先

### 薪资范围（参考）
- 校招：30-50K × 15-16薪 + 股票期权
- 社招L7：35-60K × 15-16薪 + 股票
- 社招L8：50-80K × 15-16薪 + 股票
- 美团已上市，股票可变现

### 面试侧重点
- Transformer各组件原理（RoPE/GQA/SwiGLU等）
- Scaling Laws和涌现能力
- 解码策略（Greedy/Beam/Top-K/Top-P）
- 分布式训练策略
- 算法题Medium级别 + 系统设计
- 项目经验和业务理解

## 🆕 2026年最新更新（美团 AI Agent 外卖/本地生活落地）

### 新增核心场景：外卖 AI Agent + 本地生活助手

**职责：**
- 利用大模型实现**外卖点单 Agent**：理解用户口语化需求，自动推荐并完成下单
- 构建**本地生活 Agent**：整合商户信息、用户偏好、实时优惠进行智能决策
- 负责 Agent 在**骑手调度优化**场景中的应用（多步规划 + 实时环境感知）
- 广告投放系统 AI 化：基于 Agent 的自动化广告策略优化

**2026 技术要求新增：**
- 熟悉 **美团大脑**（知识图谱）与 Agent 的融合应用
- 理解 O2O 业务场景（外卖/酒旅/到店）下的 Agent 特殊需求
- 处理 **JSON 格式错误**的鲁棒性工程：大模型输出 JSON 格式错乱时的 Fallback 机制
- **低成本 Agent**：在 Token 消耗受控前提下保证外卖点单准确率

### 美团面试新增场景题（2026）

1. **外卖点单 Agent 设计**：用户说"我想吃辣的川菜，预算30元以内，30分钟内送达"，你怎么设计这个 Agent？
2. **JSON 格式防御**：模型返回了格式错误的 JSON，你的系统怎么处理？（Retry / Structured Output / Output Parser）
3. **广告投放系统设计**：如何利用 Agent 自动优化 CPM 投放策略？
4. **成本控制**：外卖场景每天百亿次推荐，每次 Agent 调用都要消耗 Token，如何控制成本？
