# 商汤科技 AI Agent 相关岗位要求

> 最后更新：2026年6月（基于官网 JD、实习生招聘及公开信息整理）

---

## 一、公司背景

商汤科技（SenseTime）是亚洲最大的 AI 软件公司，核心优势在于**计算机视觉**和**多模态大模型**。Agent 方向主要集中在：

- **SenseNova 日日新**：多模态大模型 + Agent 能力
- **具身智能**：机器人 + LLM 融合
- **自动驾驶 AI Agent**：端到端决策系统
- **GUI Agent**：基于 InternVL 的界面操作 Agent

---

## 二、核心岗位

### 多模态大模型 & Agent 研发（实习/全职）

**职责：**
- 参与 SenseNova 多模态大模型与 Agent 系统的研发与优化
- 负责预训练、SFT、DPO、RL 等训练阶段的数据构建、训练调优与评测分析
- 构建多模态推理能力（视觉、语言、语音融合）
- 参与 Agent 能力建设：Tool Use、Planning、Router、多步推理
- 优化 Agent 在实际业务场景（零售、医疗、自动驾驶）的应用效果

**技术要求（必须）：**
- Python 精通，熟悉 PyTorch / Transformers / ms-swift / verl
- 理解大模型训练范式（预训练 → SFT → RLHF → 评估 → 部署）
- 多模态理解基础（图文/视频/语音）
- 了解 Agent 核心组件：Planning、Memory、Tool Use、多步推理

**技术要求（加分）：**
- 分布式训练经验（多卡/多机）
- 具身智能或机器人控制相关经验
- GUI Agent 或 Web Agent 研究背景
- 有 ACL/EMNLP/CVPR/ICCV 等顶会论文

---

### AI 平台基础设施工程师（大模型平台方向）

**职责：**
- 大模型平台基础设施的国产化适配开发和维护
- 基于 Kubernetes 底座的平台开发，提升系统稳定性和智能化运维
- 持续集成、部署交付（CI/CD）体系搭建
- 与算法团队配合，保障大规模训练任务的稳定运行

**技术要求：**
- Kubernetes、Docker、分布式系统
- Python + Go/Java（任一）
- 理解 GPU 资源调度（了解 NVIDIA MIG/CUDA 更佳）
- 国产芯片（昇腾、寒武纪）适配经验为加分项

## 四、面试特点

1. **算法研究深度优先**：相比纯工程岗，商汤更看重对前沿 AI 论文的理解深度
2. **多模态是门槛**：视觉模型经验显著加分
3. **项目经验核心**：训练/评测/部署全链路的实际操作经验
4. **顶会论文加分明显**：有 CV/NLP 顶会发表经历是大加分项
5. **面试流程**：技术电话面 → 2-3轮技术面（含算法题）→ 交叉面 → HR 面
